# 垃圾郵件智慧過濾系統 (期中專案)

本專案實現了一個基於機器學習的垃圾郵件（Spam）過濾系統。專案內含兩種實作路線：使用標準集成學習算法 **隨機森林（Random Forest）** 作為效能基準，以及利用純 Python 刻出的自動微分引擎進行底層邏輯驗證。

## 核心功能
- **特徵工程**：載入 `spam.csv` 後，利用 `TF-IDF` 將非結構化文字轉為高維數值特徵向量。
- **隨機森林預測**：透過多棵決策樹的投票機制（Bagging），實現高精準度的郵件分類。
- **底層架構驗證**：調用自研的 `nn0.py` 算子（`matmul`, `add`, `softmax`），搭配 Adam 優化器完成微型分類器的訓練，藉此驗證反向傳播梯度的正確性。

## 專案檔案
- `main.py`：主執行腳本（包含資料清洗、隨機森林訓練、nn0 局部優化測試）。
- `nn0.py`：純 Python 實作的標量自動微分引擎與基礎算子。
- `spam.csv`：垃圾郵件標籤數據集（ham/spam）。
